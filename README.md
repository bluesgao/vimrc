# vimrc
不用任何插件，把VIM配置成一个IDE！这个配置我已经用了好多年，希望对你有用。

## 安装
* 把vimrc文件拷贝你的home目录下，改名为.vimrc
* 在.bashrc中加入一行：
  stty -ixon
  这是为了在VIM中可以按ctrl＋s进入shell，后详
  
## 功能
配置文件简明扼要，主要功能如下：
* 使用2个空格替换一个tab
* 缩进2个空格
* 显示行号
* 自动缩进
* 自动把工作目录切换到当前打开文件所在的目录
* 高亮显示搜索关键字
* 当前行加下划线显示
* 底部状态行显示当前文件路径、行号、列号等信息
* 保存文件时自动删除行尾空格
* 启用语法高亮
* 根据文件类型自动设定缩进大小
